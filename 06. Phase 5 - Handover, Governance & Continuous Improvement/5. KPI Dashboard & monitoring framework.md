# 5. KPI Dashboard & Monitoring Framework

## 5.1 Objective
Provide ongoing visibility into IAM performance, security risk, and compliance posture.

## 5.2 KPI Categories

### Identity Lifecycle KPIs
- Onboarding SLA compliance (%)  
- Deprovisioning SLA compliance (%)  
- Orphan accounts (monthly count)  

### Access Governance KPIs
- Access review completion rate  
- SoD violations detected  
- Number of escalated access anomalies  

### Security Monitoring KPIs
- MFA adoption rate  
- Risky sign-ins detected  
- Privileged activity alerts  

### Operational KPIs
- Incidents resolved within SLA  
- Mean Time to Detect (MTTD)  
- Mean Time to Resolve (MTTR)  

## 5.3 Monitoring Tools
- **Microsoft Sentinel:** alerts, anomaly detection  
- **Entra ID Governance:** access review dashboards  
- **Power BI:** operational IAM dashboards  
- **ServiceNow:** ticketing and SLA reports  

## 5.4 Reporting Cadence
- Weekly operational dashboard  
- Monthly governance report  
- Quarterly audit summary
